# Mate (tm)

An app to make pair tutoring cool again.


## Running this app

Necesitas:

- Node
- CouchDB y Redis

Una vez instalado node y las dependencias:

```
> npm install -g bower grunt
> npm install -d
> cd public && bower install && cd ..
> grunt
> node app
```
